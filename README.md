## About The Project

Virtual Pet Kivy is a fun, interactive desktop application where you can care for your very own digital companion. Built using the Kivy framework, this project brings the classic virtual pet experience to life with a modern, responsive user interface. It's a great way to explore Kivy's capabilities for creating engaging, cross-platform applications, offering a simple yet captivating interaction loop with your virtual friend.

### Key Features
- **Interactive Pet Care:** Feed, play with, and put your pet to sleep with simple button interactions.
- **Dynamic Needs Tracking:** Monitor your pet's hunger, happiness, and energy levels in real-time.
- **State-Based Animations:** Visual feedback on your pet's current status and interactions.
- **User-Friendly Interface:** A clean and intuitive UI designed with Kivy for an enjoyable experience.

## Getting Started

To get your own virtual pet up and running, follow these simple steps.

### Prerequisites
Before you begin, ensure you have Python 3.x and pip (Python package installer) installed on your system.
*   **Python 3.x**
    ```sh
    python --version
    ```
*   **pip**
    ```sh
    pip --version
    ```

### Installation
1.  **Clone the repository:**
    ```sh
    git clone https://github.com/krishujeniya/VIRTUAL_PET_KIVY.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd VIRTUAL_PET_KIVY
    ```
3.  **Install the required Kivy library:**
    ```sh
    pip install kivy
    ```
    *(Note: Depending on your system, you might need to install Kivy's dependencies first. Refer to the [Kivy Installation Guide](https://kivy.org/doc/stable/installation/installation-linux.html) for detailed instructions for your OS.)*
4.  **Run the application:**
    ```sh
    python main.py
    ```
